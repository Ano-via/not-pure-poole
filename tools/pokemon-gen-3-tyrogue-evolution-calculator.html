巴比郎（Tyrogue）进化判定计算器

适用于《口袋妖怪 绿宝石 / 第三世代》。当巴比郎在20级升级时，比较攻击与防御的实时数值来决定进化方向。

基础值（Base）：攻击 35｜防御 35 公式：非HP = ⌊( ( (Base×2 + IV + ⌊EV/4⌋ ) × 等级 ) /100 ) + 5⌋ × 性格修正
第三世代舍入：性格修正在最终加成后再向下取整
预测结果：
当该设置在升级到选定等级（默认 20级）时生效。
攻击（Atk）：-
防御（Def）：-
如何精准控制？（小技巧）
  • 想要 飞腿郎 → 选择加攻击的性格（怕上/固执/爽朗/顽皮/勇敢中的加攻：Lonely、Adamant、Naughty、Brave），或给攻击投入更多EV。
  • 想要 快拳郎 → 选择加防御的性格（Bold、Impish、Lax、Relaxed），或给防御投入更多EV。
  • 想要 柔道郎 → 选择中性性格（Hardy、Docile、Serious、Bashful、Quirky），并在19级时把攻防调成相等（用营养剂或打对应努力值）。
  • 营养剂：力量强化剂（+10攻击EV）、铁（+10防御EV）。在 19→20 级前微调最稳。
